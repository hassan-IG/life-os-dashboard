# Quick Setup Guide — Life OS Dashboard 2026

## Web App Setup (2 minutes)

### Option A: Host Free on GitHub Pages
1. Create a free GitHub account at github.com (if you don't have one)
2. 2. Create a new repository called `life-os-dashboard`
   3. 3. Upload the `index.html` file to the repository
      4. 4. Go to Settings → Pages → Source: main branch → Save
         5. 5. Wait 1-2 minutes — your dashboard is live at `https://[your-username].github.io/life-os-dashboard/`
           
            6. ### Option B: Open Locally
            7. 1. Save `index.html` anywhere on your computer
               2. 2. Double-click to open in any browser
                  3. 3. Everything works offline — data saves to your browser
                    
                     4. ## Notion Template Setup (10 minutes)
                    
                     5. 1. Open the Notion duplication link provided with your purchase
                        2. 2. Click **"Duplicate"** in the top-right corner
                           3. 3. Choose your workspace
                              4. 4. Customize the **Life Areas** database to match your priorities
                                 5. 5. Update **Goals** with your own targets and deadlines
                                    6. 6. Add your daily **Habits** with frequencies and targets
                                       7. 7. Start tracking!
                                         
                                          8. ## Using Both Together
                                         
                                          9. The web app is your **daily cockpit** — fast, visual, satisfying for quick check-ins.
                                          10. The Notion template is your **deep system** — databases, relations, views for detailed planning.
                                         
                                          11. Use whichever fits the moment. They're designed to complement each other.
                                         
                                          12. ## Data Management
                                         
                                          13. - **Export**: Click the save icon in the top bar → Export as JSON (full backup) or CSV (spreadsheet)
                                              - - **Import**: Click the save icon → Import JSON Backup to restore from a previous export
                                                - - **Reset**: Clear your browser's localStorage to start fresh
                                                 
                                                  - ## Tips
                                                 
                                                  - - **Daily routine**: Open the dashboard, check tasks, toggle habits, log water
                                                    - - **Weekly**: Open the Weekly Review for auto-generated insights
                                                      - - **Monthly**: Export your data as a backup, review goals progress
                                                        - - **Customize**: Edit the HTML file to change colors, default data, or add sections
                                                         
                                                          - ## Troubleshooting
                                                         
                                                          - **Data disappeared?** You may have cleared your browser data. Use the import feature to restore from a backup.
                                                         
                                                          - **Charts not loading?** Make sure you have an internet connection (Chart.js loads from CDN on first visit).
                                                         
                                                          - **Dark mode stuck?** Click the sun/moon toggle in the top-right corner of the header.
